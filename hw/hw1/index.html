What is Software development?

Software development is a process involving the creation and maintenence of programs using specific programming languages. It utilizes a series of interrelated pieces of code to perfrom the task at hand.

It is an interative logical process, aimed to address a unique problem or set of business needs

Various steps include initial research, data flow design, process flow design, flow charts, technical documentation, software testing, debugging. This process is known as the software development life cycle (SDLC).
